# Dissertation_Code
This repository contains the complete R script for a Species Distribution Model (SDM) predicting Willow Warbler (Phylloscopus trochilus) habitat suitability in the UK. The model is trained on WorldClim 1998–2000 climate data and projected onto future conditions using CHELSA data for the years 2004–2005, 2009–2010, 2014–2015, and 2020.
